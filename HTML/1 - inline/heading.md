# 見出しタグ
## 見出しのタグ
見出しのタグは&lt;h1&gt;タグ～&lt;h6&gt;タグまである    
記事と同様に重要な部分を表示するために使用する  
h1タグが一番大きくh6タグが一番小さい

## 例
### 表示
![image](https://github.com/user-attachments/assets/46fce58e-cd5b-4a4b-92b4-487e8c20adda) 

### コード
```html
<h1>これは見出しです</h1>
<h2>これは見出しです</h2>
<h3>これは見出しです</h3>
<h4>これは見出しです</h4>
<h5>これは見出しです</h5>
<h6>これは見出しです</h6>
```
